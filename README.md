# Sabermetrics

A playground to learn about [sabermetrics](http://en.wikipedia.org/wiki/Sabermetrics)

## Installation

Add this line to your application's Gemfile:

    gem 'sabermetrics'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sabermetrics

## Usage
